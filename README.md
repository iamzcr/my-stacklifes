# my-stacklifes



#### 线上地址

http://iamzcr.com

#### 技术栈

服务端：golang1.22

数据库：redis6.0+，mysql8.0+

前端，vue3 ，navie ui

#### 下载项目

git clone https://github.com/iamzcr/my-stacklifes.git

#### 项目服务端运行

```
cd my-stacklifes/server/conf
vim config.yml
```

#### 服务端后台接口

```
cd my-stacklifes/server/cmd/admin
go run main.go
```

#### 服务端前台接口

```
cd my-stacklifes/server/cmd/frontend
go run main.go
```
