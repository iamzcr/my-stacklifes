下载(redis-6.2.14版本安装示例)

https://redis.io/downloads/ 或者https://github.com/redis/redis/releases

解压

wget https://github.com/redis/redis/archive/refs/tags/6.2.14.tar.gz tar xzvf 6.2.14.tar.gz

安装

yum -y install gcc gcc-c++ libstdc++-devel cd redis-6.2.14 make cd src make install PREFIX=/usr/local/redis

移动配置文件到安装目录下

cd ../ mkdir -p /usr/local/redis/etc mv redis.conf /usr/local/redis/etc

配置redis为后台启动，将daemonize no 改成daemonize yes，设置redis端口和密码，将port 6379 改成 port 你的端口，去掉requirepass注释，并设置requirepass 你的密码。

vi /usr/local/redis/etc/redis.conf

开启redis

/usr/local/redis/bin/redis-server /usr/local/redis/etc/redis.conf

登录操作redis

/usr/local/redis/bin/redis-cli -p 端口 -h 主机 //如果设置了密码，你需要执行下面命令才能操作redis auth 你的密码

关闭redis

/usr/local/redis/bin/redis-cli shutdown #指定端口和密码关闭 /usr/local/redis/bin/redis-cli -p 端口 -a 密码 shutdown
